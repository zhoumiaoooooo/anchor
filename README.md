# 锚点

为所爱之人，留下记忆的锚。

---

"她早已认不出我了，却依旧能完整哼出那首老歌，一字不差。"

---

国内有超过 1500 万个家庭，正被阿尔茨海默症裹挟。数字是冰冷的，背后是一个个具体的人——丈夫记不起妻子的名字，却清楚记得她爱吃的菜；母亲辨不出女儿的模样，可一听到几十年前那首老歌，就安静下来了。

日复一日，数年如一日。做饭、喂药，一遍遍回答同样的问题。然后某一天忽然惊醒：我忙着照料眼前这个人，却从来没有静下来，好好读懂 ta 的一生。ta 偏爱过什么，畏惧过什么，因什么开怀，又为谁落泪。

想记住的，比已经失去的，要多得多。

可大多数人翻开手机，最后只打下了一句："今天，ta 又不认得我了。"

一个人，不该被这样简单地记住。

一个人，是掌心的温度，是熟悉的声音。是夏日傍晚厨房里的油烟味，是削苹果时总爱切成兔子形状。是喊你名字时尾音拖得很长，像舍不得落地。这些细碎的片刻不值得写进任何一本传记，却会在所有往事被遗忘之后，成为一个人活过的全部证据。

于是我们做了**锚点**。

它不是病历，也不做心理测评。它只是一个安静的地方。AI 轻声问几个问题——关于一双手、一种嗓音、一段儿时旧事——你来说，它帮你妥帖地存好。不说教，不评判，也不说一句客套话。

**它无法替代陪伴，也不会催促告别。它只是一枚锚。在记忆的潮水漫上来之前，把那些珍贵的事牢牢钉住。**

推开哪扇门都可以。只写一句话也可以。什么都不记得了也没关系——只要还记得一首歌，这首歌就会被好好放在那里。

技术能抵达的最温柔的地方，就到这里了。剩下的，都是你自己的。

*记录不是为了告别。是为了证明我们爱过。*

---

## 它是怎么工作的

打开锚点，告诉它你想记录谁。然后有五扇门：

- **手** — 大小、动作、茧、温度
- **声音** — 叫你名字的方式、笑声、唱的歌
- **地方** — ta 长大的街道、气味、记忆里的空间
- **那一天** — 大的日子也好，普通的下午也好
- **还有件事** — 想说但没说的话，只存一条，不追问

每一扇门都可以随时推开。AI 会引导你说下去，每一句话都会被整理成记忆片段，存进你的记忆库。

**AI 只负责提问和整理。它不替代陪伴，也不制造告别。**

## 在自己的电脑上运行

### 你需要准备

- 一台电脑（Windows / Mac / Linux 都可以）
- [Go](https://go.dev/dl/) 1.22 或更高版本
- 一个 DeepSeek API key

### 步骤

```bash
# 1. 下载项目
git clone https://github.com/zhoumiaoooooo/anchor.git
cd anchor/server

# 2. 设置 API key
# Windows PowerShell:
$env:DEEPSEEK_API_KEY="sk-你的key"

# Mac / Linux:
export DEEPSEEK_API_KEY="sk-你的key"

# 3. 启动
go run ./cmd/server
```

打开浏览器，访问 **http://localhost:8080**。

所有数据都存储在你电脑上的 SQLite 文件里，不上传任何第三方服务器（除了调用 DeepSeek API 进行对话）。

### 可选配置

| 环境变量 | 默认值 | 说明 |
|---------|--------|-----|
| `PORT` | `8080` | 服务端口 |
| `DEEPSEEK_API_KEY` | — | **必填**，DeepSeek API 密钥 |
| `DEEPSEEK_BASE_URL` | `https://api.deepseek.com` | API 地址 |
| `DEEPSEEK_MODEL` | `deepseek-chat` | 模型名称 |
| `DATABASE_PATH` | `./data/anchor.db` | 数据库文件路径 |

## 技术

Go 后端 + 纯 HTML/CSS/JS 前端 + SQLite + DeepSeek API。

没有框架，没有构建工具，没有 npm。前端是单个 HTML 文件，后端是 Go 标准库 HTTP server。

## 关于

阿尔茨海默症是这个项目的起点，但不是它的边界。记忆的流失有很多种——疾病、时间、搬家、一个人离开你生活了很久才意识到。锚点的意思很简单：趁还抓得住，抛下去。

*那些被记住的，永远不会真的消失。*
