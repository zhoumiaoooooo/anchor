# 锚点

为所爱之人，留下记忆的锚。

## 缘起

这个项目的灵感来自阿尔兹海默症家庭。

我们不记得的事情越来越多。但不记得不等于不存在。

技术不应该代替人类去爱。它只应该做一件事：帮你记住那些你害怕会忘记的，然后退到一边。

**记录不是为了告别，而是为了证明我们爱过。**

## 设计哲学

- AI 只做记录和整理，不做陪伴和替代
- 访谈由 AI 引导，但说的每一句话都来自你
- 不聊病，不聊沉重的东西，只聊她这个人

## 技术

Go 后端 + 纯 HTML/CSS/JS 前端 + SQLite + DeepSeek API

## 运行

```bash
cd server
DEEPSEEK_API_KEY=你的key go run ./cmd/server
```

打开 http://localhost:8080
